# 银河求索具身智能教育平台

> **面向高职与应用本科的具身智能人才培养一体化解决方案**

银河求索具身智能教育平台，由银河通用机器人（Galbot）打造，面向高职院校与应用本科，专注具身智能从业人员的系统化培养。平台依托银河通用在云端仿真领域的技术积累和上百个真实商业场景的工程经验，以**理论学习 → 工程实训 → 能力认证 → 岗位就业**为核心培养链路，帮助院校快速建立具身智能方向的教学能力。

---

## 一、行业背景与政策机遇

### 1.1 具身智能上升为国家顶层战略

具身智能正从前沿技术概念快速发展为国家重点推动方向。《"十五五"规划建议》首次在五年规划级别文件中将具身智能列为重点方向（[国家发改委原文](https://www.ndrc.gov.cn/wsdwhfz/202512/t20251204_1402236.html)）；工信部印发[《人形机器人创新发展指导意见》](https://www.miit.gov.cn/jgsj/kjs/wjfb/art/2023/art_50316f76a9b1454b898c7bb2a5846b79.html)，明确到 2027 年形成具有国际竞争力的产业生态；人形机器人与具身智能标准化技术委员会已于 2025 年 12 月正式成立，下游岗位标准体系正在加速形成（[证券时报报道](https://www.stcn.com/article/detail/3591453.html)）。

### 1.2 教育政策：产教融合与 AI 赋能职教

2025 年 1 月，中共中央、国务院印发[《教育强国建设规划纲要（2024—2035 年）》](https://www.gov.cn/zhengce/202501/content_6999913.htm)，明确"加快建设现代职业教育体系"并专列"促进人工智能助力教育变革"条目；教育部职业院校信息化教学指导委员会发布《职业院校人工智能应用指引》，支持职业院校增设人工智能相关专业、与头部企业开展现场工程师培养项目（[江苏省教育厅相关行动方案](https://edu.nanjing.gov.cn/njsjyj/202506/t20250604_5577484.html)）。

然而，具身智能方向在职业院校中仍处起步阶段。全国已有 2000 余所职业院校开设人工智能相关专业，但在具身智能细分方向上，普遍面临师资储备不足、课程内容设计经验有限、缺少专业仿真实训平台等现实挑战。（[教育部官网·2025 年全国两会报道](http://www.moe.gov.cn/jyb_xwfb/xw_zt/moe_357/2025/2025_zt03/shengyin/guandian/202503/t20250310_1182150.html)）

### 1.3 产业人才需求快速增长

具身智能市场规模在 2030 年有望达到 4000 亿元，2035 年突破万亿元（国务院发展研究中心研判）。截至 2025 年 12 月，我国具身智能和机器人领域年度投资事件 744 起，融资总额 735.43 亿元。（[信通院及清华电子系《具身智能发展报告 2025》](https://mp.weixin.qq.com/s/c4XP9SW7tt6_wnB5T8xDPw)）

产业的快速发展催生了大量数据采集、模型训练与调试、现场部署等新型岗位，为具身智能方向的毕业生提供了广阔的就业空间。

> **对于院校而言，现阶段是布局具身智能专业方向的战略窗口期——政策支持已到位、产业需求明确、先行院校尚少。**

## 二、平台能力全景

平台围绕人才培养全链路，提供五个核心模块：

### 2.1 理论学习：结构化知识体系

覆盖具身智能基础理论、前沿范式（VLA / 具身大模型 / Agent 驱动机器人）、数据采集与遥操作、机器人维修等内容。

- **课程结构**：课程 → 章节 → 小节，小节支持视频、图文、附件等多种形式
- **内容层次**：从基础认知到前沿工程，适配高职至应用本科不同教学需要
- **教学资源**：配套完整教案、课件、实验脚本，教师可直接使用或结合自身教学经验灵活调整

### 2.2 工程实训：面向真实场景的动手能力

云端仿真环境与真实机器人调试双轨并行，覆盖导览、上下料、分拣、数据采集等工商业典型场景。

- **云端桌面实训**：Linux / Windows 云端桌面环境，学生打开浏览器即可操作，无需本地配置
- **仿真实训**：基于银河星坊仿真引擎，代码与机器人行为实时联动
- **真机实训**：仿真验证后迁移至真实 Galbot 机器人，完成从虚拟到现实的完整工程闭环

### 2.3 赛事对抗：以赛促学

通过竞赛激发学习动力与创新意识。赛事由独立赛事平台运营，银河可配合提供统一的后端支撑能力（赛事引擎、仿真环境）。

### 2.4 能力认证：面向岗位的能力评价

建立银河具身智能岗位能力认证体系，覆盖教师与学生双轨：

- **学生认证**：以真实岗位能力标准为锚点，为学生提供可量化、可验证的能力证明
- **教师认证**：帮助教师系统掌握具身智能领域的工程技能，获得银河通用颁发的专项能力认证

### 2.5 岗位就业：打通培养到就业的通道

银河通用拥有联营的多家数据采集工厂及上百家零售太空仓门店，持续产生数据采集、现场部署与运维等岗位需求，为通过认证的学生提供直接就业出口和优先推荐。同时，银河通用正积极推动具身智能领域的岗位能力标准建设，帮助院校形成可持续、可展示的就业成果。

## 五、合作模式与交付方式

### 5.1 合作模式

平台提供两种合作模式，适配不同院校的实际情况：

**院校直签模式**

银河通用与院校直接签约，由院校教务处等部门指定管理员，负责本院校的日常运营。适用于采购流程成熟、有明确对接人的院校。

**渠道合作模式**

银河通用通过合作伙伴触达院校，由合作伙伴人员担任院校管理员，行使相同的管理权限。适用于合作伙伴在区域内已建立成熟院校合作网络的场景。

### 5.2 交付内容

| 交付项 | 说明 |
|--------|------|
| **平台账号** | 为院校开通账号，配置授权课程和实训包范围、账号上限、有效期 |
| **课程与实训内容** | 视频、讲义、教案、实训脚本等内容预置于平台，院校管理员按需分配 |
| **硬件设备** | 支持绑定 Galbot 机器人、教学终端等硬件，满足不同采购需求 |
| **教师培训** | 为教师提供系统性培训，建立具身智能教学能力 |
| **技术支持** | 配置专属交付工程师，从部署到验收全程跟进 |

### 5.3 交付流程

```
签约 → 院校账号开通 → 课程/实训包授权 → 教师培训 → 学生账号导入 → 正式开课 → 持续运营支持
```
